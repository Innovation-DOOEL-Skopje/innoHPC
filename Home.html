How does heart monitoring work?

1. Attach an ECG sensor

Use any compatible FDA-approved or CE Mark awarded ECG sensor that receives ECG data.

2. Upload ECG data

Use manual file upload or API scripts that frequently send ECG data to ViewECG servers.

3. Start Monitoring

Login to ViewECG and monitor the ECG strips and automated findings approved by the doctor.
